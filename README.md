# 🌐 Multilingual Telegram Portfolio Bot | Aiogram 3

Сучасний та технологічний Telegram-бот-візитка з повноцінною підтримкою **мультимовності (UA / RU / EN)**, вбудованою базою даних та просунутою панеллю адміністратора.

Цей проєкт розроблений як демонстрація навичок комерційної розробки асинхронних застосунків на Python.

---

## 🛠 Технологічний стек (Tech Stack)

* **Language:** Python 3.11+
* **Framework:** Aiogram 3.x (Asynchronous Telegram Bots API)
* **Database:** SQLite3 (Робота з реляційними БД)
* **State Machine:** FSMContext (Фільтрація станів для збору та обробки ТЗ)
* **Environment:** Python-dotenv (Безпечне зберігання токенів та API ключів)
* **VCS:** Git / GitHub

---

## 🚀 Ключовий функціонал (Features)

1. **Динамічна мультимовність:** Бот автоматично визначає та зберігає обрану користувачем мову (Українська, Англійська, Російська) в базу даних. Весь інтерфейс та кнопки змінюються на льоту.
2. **База даних (Database):** Логування унікальних користувачів, збір статистики та збереження мовних пресетів.
3. **FSM Форма замовлення:** Інтерактивний збір технічного завдання (ТЗ) від потенційних клієнтів з автоматичною пересилкою адміну в приватні повідомлення.
4. **Адмін-панель (Admin Panel):**
   * Надійний захист команд за `ADMIN_ID`.
   * Моментальний перегляд статистики живих користувачів у базі.
   * Функція **розумної розсилки** повідомлень по всій базі (підтримує як чистий текст, так і медіафайли/фото).

---

## 📦 Архітектура проєкту

* `portfolio_bot.py` — Основний виконуваний файл, ініціалізація бота, логіка диспетчера та хендлерів.
* `database.py` — Модуль для роботи з базою даних SQLite (ініціалізація, читання, запис, оновлення даних користувача).
* `.gitignore` — Конфіденційний фільтр, який приховує базу даних та файли оточення `.env` від публічного доступу на GitHub.

## ⚙️ Локальний запуск (Local Deployment)

1. Клонувати репозиторій:
   ```bash
   git clone [https://github.com/MaX45165/portfolio-bot.git](https://github.com/MaX45165/portfolio-bot.git)
2. Встановити необхідні залежності:
    ```bash
    pip install aiogram python-dotenv
3. Створити файл .env у кореневій папці проєкту та заповнити його своїми даними:
    ```bash
   BOT_TOKEN=твій_токен_бота
    ADMIN_ID=твій_телеграм_id
4. Запустити проєкт:
    ```bash
   python portfolio_bot.py
